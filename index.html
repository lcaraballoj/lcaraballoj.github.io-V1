Engineering Intern

Create important demonstration models that can be used in order to show off the cybersecurity capabilities of Owl’s data diodes. This includes simulating attacks that the diodes can block as well as successfully sending packets of data. These demos utilized two Raspberry Pis sending UDP packets of sensor data to each other with the XDE Radium in the middle to filter out the packets. Along with this demo came close analysis of the documentation of the XDE Radium and the creation of documentation for the demos in order to provide better insight and support of the usability of the XDE Radium. The use of MQTT packages was also explored on a clients order to receive MQTT packages and have them automatically converted to a CSV file in order to store its data in a FTP server.
